# GPA Calculator

First Flutter Project that helps calculate GPA for FAST Student(other Universities too if they have the same grading and GPA system).
how are you doing?

### FAST Grading System

A+/A -> 4.00
A-   -> 3.67
B+   -> 3.33
B    -> 3.00
B-   -> 2.67
C+   -> 2.33
C    -> 2.00
C-   -> 1.67
D+   -> 1.33
D    -> 1.00
F    -> 0
